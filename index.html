Series introduction

Part 1 - IoT Server

In Part 1 we will build a fully functional IoT server and attach a lamp you can control from a webpage.

Hardware needed: Note these are not affiliate links - find cheapest price you can

  • Raspberry Pi
  • IoT Power Relay - Optional

Software needed:

  • Visual Studio Code
  • Raspberry Pi Imager

Steps:

  • Burn OS to SD Card
  • Boot Raspberry Pi - Update
  • Install Nodejs
  • Connect LED to Pi
  • Run Local Tests from terminal
  • Clone code from git
  • Modify Code in Visual Studio Code
  • Run Local Tests from web page
  • Install Dataplicity
  • Install PM2
  • Install NginX
  • Have fun with light
